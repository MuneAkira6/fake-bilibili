### 涉及技术
HTML+CSS+JavaScipt基础，未采用框架

### 亮点
- 用js实现轮播图效果。每一秒自动滚动一次，鼠标悬停其上则暂停滚动，鼠标移开继续滚动。鼠标点击轮播图下方小点，则跳转到对应的图片。
- 在第一个轮播图右侧的六个推广视频（class="info"）中，实现了鼠标悬停，跳出视频详细信息的效果。
